# Web Development Projects

This repository contains small web development projects created as part of university assignments and independent practice.

The goal of these projects is to practice basic **front-end web development** using HTML, CSS and JavaScript.

## Note
These projects that I am developing are in Croatian language.

## Projects

### Ski Trip Cost Calculator

This is a small interactive website that estimates the approximate cost of a ski trip.

Users can choose different options such as:

* ski resort
* equipment
* accommodation
* food

Based on the selected options, the website calculates an approximate total cost of the trip.

This project is still **work in progress**, but the current version of the website can already be viewed in the repository.

---

### AI Tools Web App

This project is a website that presents several artificial intelligence tools.

The goal of the website is to demonstrate:

* descriptions of AI tools
* tables and structured information
* embedded media
* interactive elements

This project is **currently under development** and requires additional adjustments (server setup, fixing links between HTML, CSS and JavaScript files).
Because of that, the website cannot be viewed directly at the moment.
