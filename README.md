# Web Development Projects

This repository contains several small web development projects created as part of university assignments and independent practice.

The projects focus on practicing basic front-end development skills using HTML, CSS and JavaScript.

## NOTE
Projects are in Croatian language.

## Projects

### AI Tools Web App

A simple website presenting several artificial intelligence tools.

Features include:

* descriptions of selected AI tools
* comparison table
* embedded video
* interactive form
* basic navigation between pages

### Ski Trip Cost Calculator

A small interactive website that helps users estimate the cost of a ski trip.

Users can select:

* ski resort
* equipment
* accommodation
* food options

The application then calculates an approximate total price for the trip.

## Technologies Used

* HTML
* CSS
* JavaScript
